# Diagrams

This folder stores visual documentation diagrams used by the README and supporting docs.

## Current Diagrams

| File | Purpose |
| --- | --- |
| `architecture-diagram.png` | High-level system architecture for the PyQt5 DSP application. |
| `workflow-diagram.png` | End-to-end application workflow from user input to PDF export. |

## Guidelines

- Keep diagrams in PNG format for reliable GitHub rendering.
- Match the dark purple visual style used by the application.
- Use lowercase kebab-case filenames.
- Update `docs/architecture.md`, `docs/workflow.md`, and the root `README.md` when diagrams change.
