# Samples

This folder contains sample files used to test and demonstrate the application.

## Images

- `images/demo/` contains small noisy demo images.
- `images/bsds/` contains image samples organized into `train/` and `test/` folders.

## Audio

- `audio/noizeus-dataset/` contains WAV samples from the NOIZEUS-style speech noise dataset used for audio filtering demonstrations.

The application can load images and WAV files from any location; these samples are provided only for testing and demonstration.
