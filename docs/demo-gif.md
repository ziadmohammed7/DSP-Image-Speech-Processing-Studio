# Demo GIF Guide

The current demo GIF is saved at:

```text
assets/demo/app-demo.gif
```

It was assembled from real captured application states. For a more natural public showcase, record a short manual interaction after publishing the repository.

## Recommended GIF Content

Capture a 10 to 15 second workflow:

1. Launch the application.
2. Load a sample image.
3. Apply a Gaussian or median filter.
4. Switch to the audio tab.
5. Show waveform and spectrogram visualization.
6. Export a PDF report.

## Recommended Tools

| Tool | Use |
| --- | --- |
| ScreenToGif | Best Windows option for short polished GIFs. |
| ShareX | Good for quick screen capture and GIF/video export. |
| OBS Studio | Best for high-quality video, then convert a short clip to GIF. |

## Recommended Capture Settings

- Resolution: 1280x720 or 1366x768.
- Duration: 10 to 15 seconds.
- Frame rate: 10 to 15 FPS for GIF, 30 FPS for video.
- Format: GIF for README preview, MP4 for LinkedIn or release pages.
- Filename: `assets/demo/app-demo.gif`.
