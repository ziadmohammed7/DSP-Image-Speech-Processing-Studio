# Image Processing Screenshots

Recommended captures:

| Filename | What To Capture |
| --- | --- |
| `image-tab.png` | Image Processing tab with input and output panels visible. |
| `noise-reduction.png` | Gaussian or median filter applied to a noisy grayscale image. |
| `edge-detection.png` | Canny or Sobel edge detection result. |
| `histogram-equalization.png` | Contrast enhancement before/after example. |

Use the demo files in `samples/images/demo/` for repeatable captures.
