# Documentation

This folder contains academic and generated documentation for Image & Speech Processing Studio.

## Contents

| Folder | Contents |
| --- | --- |
| `report/` | Final DSP project report in PDF and editable DOCX formats. |
| `presentation/` | Project presentation in PDF and PPTX formats. |
| `generated-reports/` | Example PDF reports exported by the application. |
| `legacy/` | Original README and instruction text files preserved from the submitted folder. |
| `use-cases/` | Reserved for deeper future case-study writeups. |

## Main Files

- Report PDF: `report/digital-signal-processing-report.pdf`
- Report DOCX: `report/digital-signal-processing-report.docx`
- Presentation PDF: `presentation/image-speech-processing-studio.pdf`
- Presentation PPTX: `presentation/image-speech-processing-studio.pptx`
- Screenshot guide: `screenshots.md`
- Use-case matrix: `use-cases.md`
- Workflow explanation: `workflow.md`

Generated PDF report examples are useful for showing the application's automatic reporting output.

## Portfolio Documentation

- [`screenshots.md`](screenshots.md) explains every screenshot needed for a polished README and release page.
- [`use-cases.md`](use-cases.md) maps the application features to real-world academic, image-processing, audio-processing, reporting, and deployment scenarios.
- [`workflow.md`](workflow.md) explains the end-to-end flow from user input to PDF export.
