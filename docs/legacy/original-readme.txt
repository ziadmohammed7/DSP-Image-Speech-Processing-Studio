Project Description:
This project is a desktop application that demonstrates practical Digital Signal Processing (DSP) concepts through image and speech processing.

The application allows users to:
- Apply image processing filters (Noise Reduction, Edge Detection, etc.)
- Apply speech/audio processing filters (Noise Removal, Low-pass Filtering, Electrical Hum Removal)
- Visualize signals using waveforms and spectrograms
- Export a detailed PDF report containing processing steps, parameters, and results

The project was developed to bridge theoretical DSP concepts with real-world applications using an interactive graphical user interface (GUI).

--------------------------------------------------

Main Features:
- Image Processing (Gaussian, Median, Edge Detection)
- Speech Processing (Low-Pass Filter, Noise Reduction, 50Hz Electrical Hum Removal)
- Real-time parameter control
- Input vs Output visualization
- Automatic PDF report generation
- Standalone Windows executable (no Python installation required)
