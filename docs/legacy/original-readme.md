\# 📌 Project Overview



Image \& Speech Processing Studio is a desktop application that demonstrates the practical implementation of Digital Signal Processing techniques.



The application allows users to process both images and speech through an intuitive graphical interface while visualizing the results instantly.



\---



\# ✨ Features



\## 🖼️ Image Processing



\- Gaussian Filter

\- Median Filter

\- Bilateral Filter

\- Average Filter

\- Sobel Edge Detection

\- Laplacian Filter

\- Canny Edge Detection

\- Grayscale Conversion

\- Histogram Equalization

\- Adjustable Kernel Size

\- Live Preview

\- Save Processed Image



\---



\## 🎤 Speech Processing



\- Low-Pass Filter

\- High-Pass Filter

\- Band-Pass Filter

\- Band-Stop Filter

\- Speech Noise Reduction

\- Remove 50 Hz Electrical Hum

\- Waveform Visualization

\- Spectrogram Visualization

\- Adjustable Cutoff Frequency

\- Adjustable Filter Order

\- Save Processed Audio



\---



\# 🖥️ Graphical User Interface



The application was developed using \*\*PyQt5\*\* to provide a modern desktop experience.



Main GUI components include:



\- Image Viewer

\- Audio Viewer

\- Waveform Display

\- Spectrogram Display

\- Processing Controls

\- Live Parameter Adjustment

\- Export PDF Report
